# Histórico de Mudanças

## 30/06/2026

### ✨ Novidades
- **Descrição nas tarefas** — Agora dá para escrever um textinho explicando melhor cada tarefa. Esse textinho aparece logo abaixo do título, com letra menor e mais clarinha.
- **Campo de descrição** — Do lado do campo onde você digita o nome da tarefa, tem um espaço maior para escrever a descrição.
- **Descrição pode ser editada** — Se você clicar para editar uma tarefa, aparece o campo com a descrição que já estava lá, e você pode mudar.
- **Descrição na exportação** — Quando você baixa a lista de tarefas para um arquivo de planilha, a descrição também vai junto.
- **Descrição na importação** — Se você carregar um arquivo de planilha com as tarefas, o sistema também lê a descrição de cada uma.
- **Descrição na comunicação com o servidor** — Tudo que o sistema envia ou recebe também inclui a descrição, então nada se perde.

### ⚙️ Melhorias
- **Explicação visual do projeto** — O README agora tem um desenho mostrando como o sistema funciona, com passo a passo desde a abertura de uma solicitação de mudança até a atualização automática deste histórico.

## 29/06/2026

### 🔒 Entrada no Sistema
- **Tela de login** — Agora tem uma tela onde você digita seu e-mail e uma senha para entrar. Só quem tem permissão consegue usar o quadro de tarefas.
- **Proteção do sistema** — As informações são protegidas com um código de segurança que dura 24 horas. Quando esse código vence, o sistema sai sozinho e volta para a tela de login.
- **Botão Sair** — Você pode clicar em "Sair" para encerrar seu acesso a qualquer momento.
- **Entrada automática** — Se você já entrou antes e o código de segurança ainda é válido, o sistema te leva direto para o painel, sem precisar digitar e-mail e senha de novo.

### 📋 Quadro de Tarefas
- **Quadro visual** — A listinha virou um quadro com três colunas: A Fazer, Fazendo e Pronto. Assim fica fácil ver o que está esperando, o que está sendo feito e o que já terminou.
- **Arraste e solte** — Para mudar uma tarefa de coluna, é só pegar o cartão com o mouse e arrastar para o lugar certo.
- **Botões para mover** — Cada cartão também tem setinhas ← e → para levar a tarefa para a coluna ao lado, sem precisar arrastar.
- **Campo de situação** — Cada tarefa mostra em qual etapa ela está (a fazer, fazendo ou pronto), em vez do antigo "sim/não" de concluído.
- **Funciona no celular** — Em telas pequenas, o quadro se ajusta sozinho para mostrar uma coluna por vez, facilitando o uso no celular.

### 📁 Importar e Exportar
- **Importar arquivo** — Você pode carregar várias tarefas de uma vez clicando em "Importar" e escolhendo um arquivo no computador.
- **Exportar arquivo** — Dá para baixar sua lista de tarefas como um arquivo que abre no Excel ou em outros programas de planilha.
- **Importar e exportar com situação** — Tanto na hora de carregar quanto de baixar, o sistema considera em qual coluna do quadro cada tarefa está.

### 🛠️ Melhorias Internas
- **Aplicativo de tarefas** — Foi criado um aplicativo onde você pode adicionar, ver, editar e excluir tarefas.
- **Guarda automática na nuvem** — As tarefas são salvas automaticamente no servidor, então você não perde nada quando fecha a página.
- **Manual do projeto** — O arquivo README foi criado e atualizado com explicações sobre o projeto e como usar.
- **Atualização automática** — Foram criados processos para gerar este histórico de mudanças automaticamente sempre que algo novo for lançado.
